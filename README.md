# 📓 Название хранилища

Краткое описание — что это за хранилище и зачем оно нужно.
Например: личная база знаний, заметки по учёбе/работе, second brain и т.д.

## 📁 Структура

```
.
├── 00-Inbox/        # входящие заметки, ещё не разобранные
├── 01-Projects/     # заметки по активным проектам
├── 02-Areas/        # области ответственности (здоровье, финансы...)
├── 03-Resources/    # справочные материалы, статьи, книги
├── 04-Archive/       # завершённые/неактуальные заметки
└── Templates/       # шаблоны заметок
```

> Структуру можно менять под себя — это просто пример на основе PARA.

## ⚙️ Как пользоваться

1. Склонировать репозиторий:
   ```bash
   git clone https://github.com/username/repo-name.git
   ```
2. Открыть папку как хранилище в Obsidian (`Open folder as vault`).
3. Включить нужные плагины в настройках.

## 📌 Заметки

- Хранилище синхронизируется вручную / через Git.
- Часть заметок может быть приватной и не публиковаться (см. `.gitignore`).
